# Different between Flex & Grid system Layout
# Justify
ইহা কাজ করে মেইন এক্সিস তথা রো বরাবর । সুতরাং ওয়াইডথ নির্ধারণ না করলে ইহা কাজ করবেনা। 
Justify content  -> In flex  
Justify items     -> In grid
